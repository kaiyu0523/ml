# 車道線影像辨識 

#目的：
為了實現自走車在道路上可沿著車道移動且不會偏離車道，因此撰寫一道路車道線影像辨識的程式。

實現：
此程式會針對行車紀錄器的影像進行辨識，並且標記出(1)車道區域(2)道路中線(3)車道線

優化：
在初版的程式中，當無法偵測到車道線時，最後的偵測結果會出現問題。
因此，我在程式碼當中加入記憶功能，即當程式無法偵測車道線時，會將上一幀所偵測到的結果當作此次的結果，直到再次偵測到車道線時才會更新為新的結果。
這樣做是為了針對可能會發生車道線不清楚的問題，透過這樣的方式，可以讓偵測結果維持車輛移動的原方向，不會讓偵測結果有巨大偏移或是其他意料之外的狀況發生。
