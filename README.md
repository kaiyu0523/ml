# 車道線影像辨識 

## 🎯 目的
為了實現自走車在道路上能沿著車道移動且不會偏離車道，因此撰寫了一個 **車道線影像辨識程式**。

---

## ⚙️ 實現
此程式會針對 **行車紀錄器影像** 進行辨識，並標記出以下三項資訊：

1. 🟩 **車道區域**
2. 🔴 **道路中線**
3. ⚪ **車道線**

---

##  優化
在初版程式中，當無法偵測到車道線時，偵測結果會出現問題。  
為了解決這個狀況，程式碼加入了 **記憶功能**：

- 當無法偵測到車道線時，程式會使用 **上一幀的偵測結果** 作為本次輸出。  
- 當再次偵測到新車道線時，才會更新結果。  

這樣的方式可以：
- 在車道線模糊或短暫消失時，**維持穩定的方向**  
- 避免因影像品質造成 **偵測偏移或跳動**
